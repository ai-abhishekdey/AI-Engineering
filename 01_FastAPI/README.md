# FastAPI

## Goal

Learn how production AI systems expose machine learning models as scalable web services.

## Learning Outcomes

After completing this module, we will be able to:

- Explain how HTTP works
- Design REST APIs
- Build FastAPI services
- Validate requests using Pydantic
- Structure production-ready FastAPI projects
- Understand asynchronous programming
- Deploy AI inference APIs
